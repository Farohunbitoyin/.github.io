My Data Science Projects

These are some of my most interesting projects which include different Machine Learning Algorithms, Sentiment Analysis, NLP, Time series Forecasting and, others.

Heart Attack Prediction

The aim of this project is to predict the chances of a patient having heart attack based on the health data gathered. Five different classification algorithms were used for comparision.

Daily Bike Rentals

Prediction of daily bike rental based on the environmental and seasonal settings. The aim of this project is to predict daily bike rentage using Linear Regression algorithm. Hyperparameter was done to improve the result.

Sentimental Analysis of YouTube Comments

This is a Natural Language Processing project where comments on youtube video was scrapped and analyzed. I used Wordcloud to show the positive and negative reviews, emoji analysis was also performed.

Time Series Analysis - Forecasting of female birth in Carlifornia

The detailed jupyter notebook contain three different datasets which I used to display my competencies in timeseries analysis. The dataset includes; total number of female births recording in California,US airline dataset & daily minimum temperature. Few of the points covered are data visualization,decomposition of a series, differencing , lag, use of ARIMA & SARIMA model etc.

Loan Defaulter Analysis

The loan default dataset was analysed using Decision Tree and Random Forest. It was deduced that most of the people have CreditScore between 550 and 750, females are more defaulters than males. Conclusively, it was seen that Random Forest is giving low bias and low variance error among other models.
